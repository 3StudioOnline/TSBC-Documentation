## Get Layer 2 Configuration

`Get Layer 2 Configuration` is a helper function that Gets configuration data for a specific Layer2 network needed
to send HTTP calls to <Step text="1"/>.

![Get Layer 2 Configuration](_static/img/L2-config.png)

This function returns a *Struct* that contains:
* <Step text="2"/> Network's RPC URL endpoint.
* <Step text="3"/> Network's name.

Current networks supported for now are:
* Loopring
* ImmutableX
